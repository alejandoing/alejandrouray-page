Python Algorithms es un proyecto de pequeños programas que se ejecutan en el terminal de Windows, Linux o Mac. Es una implementación pensada para poner en práctica diferentes conceptos básicos del lenguaje, como condicionales, ciclos, clases, funciones, etc. Entre los programas desarrollados destaca el software de criptografía que se encarga de realizar un sencillo encriptado de cadenas de texto a codigo ASCII. Por otro lado, también sobresale el programa que emula el popular juego del "ahorcado", en el cual tienes como objetivo adivinar la palabra secreta en una cantidad limitada de intentos.

Descripción

  • Febrero 2018

  • En Desarrollo

Experta Monitoring es un proyecto de programación backend-hardware. Tiene como propósito ofrecer una solución que permita obtener datos referenciales de temperatura, humedad, iluminación y ruido de un entorno laboral para la posterior toma de decisiones. Los datos se recuperan a través de sensores conectados a dispositivos como Raspberry Pi y Arduino. Estos dispositivos, a su vez, establecen un puente con el servicio Lambda de Amazon Web Services para reportar a un dashboard central la información obtenida en tiempo real.

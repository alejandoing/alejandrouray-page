Previsalud es un sistema de administración de polizas de seguros. Permite realizar diferentes operaciones como generar una Carta Aval, procesar una Clave de Emergencia, tramitar un Reembolso, etc. Además, contiene un módulo de farmacia que establece una conexión con un API para obtener información en tiempo real de inventario de medicamentos. Todos los días se realizan cientos de operaciones en la plataforma. Es un sistema con alta concurrencia de usuarios las 24hs del día.
